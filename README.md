# 🎓 School History & Courses Quiz

An interactive, web-based quiz application designed to test your knowledge about the school's history, courses, and legacy. Built entirely with Vanilla web technologies (HTML, CSS, JS) as part of an academic project.

## 🚀 Live Demo
**Play the Quiz Here:* https://adityax-07.github.io/School-Quiz-Project/

## ✨ Features
- **Interactive UI:** Clean, intuitive, and fully responsive design across all devices.
- **Smooth Animations:** Engaging CSS transitions and keyframes (fade-ins, pulsing timer, sliding feedback sections).
- **Instant Feedback:** Real-time correct/incorrect validation with detailed explanations for every question.
- **Dynamic Timer:** 15-second countdown per question with a visual and stress-inducing "pulse" warning when time is low.
- **Smart Randomization:** Uses the Fisher-Yates algorithm to randomly shuffle questions and answer options every time the quiz is started.
- **Progress Tracking:** Live dynamic progress bar and question counter.
- **Accessibility (a11y):** Keyboard navigable and screen-reader friendly using native HTML semantics and ARIA attributes.
- **Result Sharing:** Native Web Share API integration to easily share final scores with friends.

## 🛠️ Technologies Used
- **HTML5:** Semantic structure and layout.
- **CSS3:** Custom styling, CSS variables, Flexbox layout, and CSS animations (No external frameworks).
- **JavaScript (ES6+):** DOM manipulation, state management, timer logic, and interactivity.
